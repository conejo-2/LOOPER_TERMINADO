LOOPER 2 - INSTALACIÓN Y USO

1. REQUISITOS:
   - XAMPP o servidor web con PHP y MySQL
   - Navegador web moderno

2. INSTALACIÓN:
   - Copia la carpeta "looper-2" en tu directorio htdocs de XAMPP
   - Inicia Apache y MySQL desde XAMPP
   - Importa el archivo database.sql en phpMyAdmin para crear la base de datos
   - Accede a http://localhost/looper-2/LOOPER_MENU.html

3. ESTRUCTURA DE ARCHIVOS:
   - LOOPER_MENU.html - Menú principal
   - LOOPER_LOGIN.html - Pantalla de login
   - LOOPER_REGISTRO.html - Pantalla de registro
   - LOOPER_JUEGO.html - El juego
   - LOOPER_TERMINAR.html - Game Over
   - login.php - Procesamiento de login
   - registro.php - Procesamiento de registro
   - style.css - Todos los estilos
   - database.sql - Estructura de base de datos

4. FUNCIONALIDADES:
   - Sistema completo de login/registro
   - Juego de serpientes funcional
   - Validaciones de formularios
   - Diseño responsive
   - Base de datos MySQL

5. NAVEGACIÓN:
   - Menú → Login → Juego → Game Over → Menú
   - Todos los botones funcionan correctamente
   - Mensajes de error/éxito incluidos

¡Listo para usar!
